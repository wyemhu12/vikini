# Coding Standards

## TypeScript Enforcement

- **Strict TypeScript**: All new files MUST be created as `.ts` or `.tsx`.
- **No JavaScript**: The creation of `.js` or `.jsx` files is strictly prohibited in this project.
- **Type Checking**: All changes must pass `npm run type-check`.

## File Naming

- React Components: `PascalCase.tsx`
- Hooks: `useCamelCase.ts`
- Utilities: `camelCase.ts`
