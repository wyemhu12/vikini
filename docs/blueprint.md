# **App Name**: Momentum AI

## Core Features:

- User Input: Text field for users to enter their questions and prompts for the AI.
- AI Response Generation: Real-time display of the AI-generated response based on user input. The tool should incorporate and respond based on past turns of the conversation.
- New Chat Initiation: Button or link to start a new conversation, clearing the current chat history.
- Chat History: Display a scrollable, reverse-chronological list of prior conversations for users to review past interactions.
- Persistent Storage: All conversations are persisted in the Cloud SQL database.
- Auto-Title: Automatically generate a title for the conversation based on the content.

## Style Guidelines:

- Primary color: Muted blue (#79A9D9) to convey trust and intelligence.
- Background color: Light gray (#F0F4F7) for a clean, minimalist aesthetic.
- Accent color: Soft purple (#A69CAC) as a complementary hue for interactive elements.
- Body and headline font: 'Inter' sans-serif font for a modern, neutral appearance.
- Use minimalist line icons for chat controls, user profiles, and settings.
- Clean and spacious layout with a focus on readability and user experience.
- Subtle transition animations for displaying AI responses.
